# Data Breach

**Group 9A — GBDA302 Midterm Game**

A cyberpunk platformer representing the experience of low vision through environmental design.

## How to Play

Open `index.html` in Chrome. All libraries are included locally in `libraries/`.

| Control | Action |
|---------|--------|
| Arrow keys / WASD | Move |
| Space / Up / W | Jump |
| Space / Up / W (mid-air) | Double jump |
| Hold F (while still on ground) | Echolocation — reveals nearby platforms, 3.5s cooldown |
